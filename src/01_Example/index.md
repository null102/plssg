# Example

A sub-site. Its directory name `01_Example/` becomes the URL `/Example/`;
the `NN_` prefix only controls ordering in the top nav.

- Put more `.md` files next to this one — they show up in the sidebar.
- Nested directories nest in the sidebar too.
- Drop images into `assets/img/`, downloads into `assets/file/`.
- Reference them with `![alt](assets/img/foo.png)` or `[doc](assets/file/bar.pdf)`.
